# docker-spn-template
Ce repo est un template de projet Symfony minimal prêt à l’emploi. Stack moderne et légère basée sur Docker : - PHP / Symfony : squelette Symfony. - Nginx : serveur web configuré pour servir l’application Symfony. - PostgreSQL : base de données relationnelle. - Docker Compose : orchestration des services (app, web, db) avec une seule commande.
